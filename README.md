# MAITOK (Tikly)

TikTok 댓글 분석 + 자동 대댓글 AI 서비스

## 개요

- **브랜드명:** Tikly
- **프로젝트명:** MAITOK
- **핵심 기능:** TikTok 댓글 감성분석, 자동 대댓글 생성, 셀러 리포트
- **타겟:** TikTok Shop 셀러 (한국/베트남)

## Tech Stack

- TikTok Content Posting API
- OpenClaw Skill (댓글 분석 + 응답 자동화)
- AI 감성분석 (LLM 기반)
