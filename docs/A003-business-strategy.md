# A003 - Tikly 사업화 전략

## 1. 비즈니스 모델

### 1.1 수익 모델: Freemium + 사용량 기반 하이브리드

| 구분 | Free | Pro | Business |
|------|------|-----|----------|
| **월 가격 (한국)** | ₩0 | ₩29,000 | ₩79,000 |
| **월 가격 (베트남)** | 0₫ | 199,000₫ (~$8) | 499,000₫ (~$20) |
| **댓글 분석** | 100건/일 | 무제한 | 무제한 |
| **자동 대댓글** | 20건/일 | 500건/일 | 2,000건/일 |
| **AI 감성분석** | 기본 (긍/부/중) | 상세 (6단계) | 상세 + 커스텀 |
| **언어 지원** | 한국어 or 베트남어 | 한/영/베 | 한/영/베 + 태/인니 |
| **대댓글 톤 커스텀** | ✗ | ✓ | ✓ + 브랜드 보이스 |
| **대시보드** | 기본 | 고급 분석 | 고급 + API 연동 |
| **계정 수** | 1개 | 3개 | 10개 |
| **우선 지원** | ✗ | 이메일 | 전담 매니저 |

### 1.2 가격 전략 근거

- **한국**: SaaS 평균 ARPU $25-40/월 수준에 맞춤. 뷰티/패션 셀러 월 매출 대비 1% 미만 비용
- **베트남**: 구매력 기준 한국의 30-40% 수준으로 차등. 베트남 SaaS 시장 가격 감수성 반영
- **초과 사용량**: Pro 기준 대댓글 500건 초과 시 건당 ₩30 / 500₫ 추가 과금

### 1.3 추가 수익원 (Phase 3+)

- **셀러 인사이트 리포트**: 경쟁사 댓글 트렌드 분석 (월 ₩49,000)
- **에이전시 플랜**: 다수 셀러 관리용 화이트라벨 (월 ₩199,000~)
- **API 액세스**: 외부 시스템 연동 (건당 과금)

## 2. GTM (Go-To-Market) 전략

### 2.1 초기 고객 확보: MAIBEAUTY 네트워크 활용

**Phase 1 타겟 (0-2개월):**
- MAIBEAUTY 기존 베트남 뷰티 셀러 네트워크 → 첫 50명 베타 유저
- 한국 뷰티/패션 TikTok Shop 셀러 → 카페/커뮤니티 모집 50명

**구체 실행:**
1. MAIBEAUTY 거래처에 Tikly 무료 체험 제공 (3개월 Pro 무상)
2. 베타 유저 피드백 → 제품 개선 루프
3. 성공 사례 → 케이스 스터디 제작

### 2.2 마케팅 채널

| 채널 | 전략 | 예상 CAC |
|------|------|----------|
| **TikTok 자체** | Tikly로 관리한 셀러의 실제 성과 before/after 영상 | ₩5,000 |
| **셀러 커뮤니티** | 네이버 카페 (스마트스토어/TikTok 셀러), 베트남 Shopee/TikTok 포럼 | ₩8,000 |
| **인플루언서** | 뷰티/패션 마이크로 인플루언서 (10K-100K) 리뷰 | ₩15,000 |
| **콘텐츠 마케팅** | "TikTok 셀러 댓글 관리 가이드" SEO 블로그 | ₩3,000 |
| **파트너십** | TikTok Shop 에이전시/MCN 제휴 | ₩10,000 |

### 2.3 베타 테스트 전략

```
Week 1-2: 클로즈드 베타 (MAIBEAUTY 셀러 20명)
  → 핵심 기능 안정성 검증
Week 3-4: 확장 베타 (초대 코드, 50명 → 100명)
  → 다양한 카테고리/언어 테스트
Week 5-8: 오픈 베타 (무료 가입, 목표 500명)
  → 스케일 테스트 + 바이럴 측정
```

### 2.4 바이럴 요소 설계

1. **"Powered by Tikly" 워터마크**: 자동 대댓글에 미묘한 브랜딩 (옵트아웃 가능)
2. **초대 보상**: 친구 초대 시 양쪽 Pro 7일 무료
3. **성과 공유 카드**: "이번 주 Tikly로 댓글 응답률 340% 증가!" 공유용 이미지 자동 생성
4. **셀러 리더보드**: 응답률/감성 점수 랭킹 (옵트인)

## 3. 경쟁 우위 & 차별화

### 3.1 경쟁 환경

| 경쟁사 | 특징 | Tikly 대비 약점 |
|--------|------|-----------------|
| **Chatfuel** | 멀티플랫폼 챗봇, TikTok DM 자동화 | 댓글 감성분석 미지원, 높은 가격 ($15+/월) |
| **Kommo** | CRM 기반 TikTok 연동 | 과도한 기능, 셀러 특화 아님 |
| **수동 관리** | 셀러 직접 댓글 응답 | 시간 비용 높음, 24시간 대응 불가 |
| **일반 AI 도구** | ChatGPT 등으로 수동 복붙 | 자동화 없음, TikTok 맥락 이해 부족 |

### 3.2 Tikly만의 강점

1. **TikTok Shop 셀러 전용**: 범용 도구가 아닌 셀러 워크플로에 최적화
2. **AI 감성분석 + 자동 대댓글**: 단순 키워드 매칭이 아닌 맥락 이해 기반 응답
3. **한국/베트남 로컬라이제이션**: 
   - 한국어 존댓말/반말 자동 판별
   - 베트남어 방언/슬랭 대응
   - 현지 결제 수단 (카카오페이, MoMo, ZaloPay)
4. **MAIBEAUTY 시너지**: 뷰티 도메인 전문 지식 내장, 화장품 성분/효능 관련 자동 응답
5. **합리적 가격**: 베트남 시장 맞춤 가격 (경쟁사 대비 50-70% 저렴)

### 3.3 기술적 차별화

- **OpenClaw 기반 아키텍처**: 유연한 AI 에이전트 파이프라인, 빠른 기능 확장
- **멀티 LLM 지원**: GPT-4o, Claude, Gemini 등 선택 가능 → 비용/품질 최적화
- **실시간 학습**: 셀러별 응답 패턴 학습 → 사용할수록 정확도 향상

## 6. 리스크 & 대응

### 6.1 TikTok API 정책 변경

| 리스크 | 영향 | 대응 |
|--------|------|------|
| API 접근 제한/폐쇄 | 서비스 중단 가능 | 크롤링 폴백 + 멀티 플랫폼 확장으로 의존도 분산 |
| Rate limit 강화 | 자동 대댓글 속도 저하 | 큐 기반 비동기 처리, 우선순위 스케줄링 |
| TikTok 자체 도구 출시 | 시장 잠식 | 니치 기능(감성분석, 브랜드보이스)으로 차별화 유지 |

### 6.2 자동 댓글 스팸 판정

| 리스크 | 영향 | 대응 |
|--------|------|------|
| 계정 제한/정지 | 셀러 신뢰 하락 | 인간적 응답 생성 (다양한 표현, 지연 시간 랜덤화) |
| 동일 패턴 탐지 | 대댓글 차단 | 응답 다양성 엔진, 템플릿 로테이션 |
| TikTok ToS 위반 | 법적 리스크 | ToS 지속 모니터링, 컴플라이언스 팀 운영 (Phase 3+) |

**핵심 완화 전략:**
- 대댓글 속도 제한 (5-30초 랜덤 지연)
- 동일 문구 반복 방지 (LLM 기반 자연스러운 변형)
- 셀러별 일일 한도 설정 (플랫폼 가이드라인 준수)

### 6.3 경쟁자 진입

| 리스크 | 영향 | 대응 |
|--------|------|------|
| 대형 SaaS 진입 (Hootsuite 등) | 가격/기능 경쟁 | 로컬 시장 집중, 가격 우위 유지 |
| 유사 스타트업 등장 | 시장 분할 | 선점 효과 + 셀러 데이터/학습 모아트 |
| TikTok 공식 도구 | 기본 기능 무료화 | 고급 분석/커스텀 영역으로 포지셔닝 |

### 6.4 기타 리스크

- **AI 환각**: 잘못된 제품 정보 응답 → 셀러별 팩트 DB 구축으로 대응
- **개인정보**: 댓글 데이터 수집/저장 → GDPR/PDPA 준수 설계
- **환율 변동**: 베트남 VND 가격 → 분기별 가격 리뷰
