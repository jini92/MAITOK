|------|-----|------|
| 동남아시아 (전체) | ~$22B | 전체의 ~2/3, 인도네시아·태국 주도 |

### 성장 동인
- **숏폼 콘텐츠 → 구매 전환**: 검색이 아닌 발견 기반 쇼핑
- **라이브커머스**: 동남아 시장 핵심 드라이버
- **크리에이터 어필리에이트**: 셀러-크리에이터 협업 모델 확산
- **Z세대 소비**: TikTok 주 사용층의 높은 충동 구매 성향

## 2. TikTok 셀러 댓글 관리 페인포인트

### 주요 문제점

| 페인포인트 | 영향 | 심각도 |
|------------|------|--------|
| **댓글 폭주**: 인기 영상 수백~수천 개 댓글 | 수동 확인 불가, 구매문의 누락 | 🔴 높음 |
| **구매의도 놓침**: "얼마에요?" 류 댓글 미응답 | 직접 매출 손실 | 🔴 높음 |
| **부정 댓글 방치**: 클레임/불만 조기 대응 실패 | 브랜드 이미지 훼손 | 🔴 높음 |
| **다국어 혼재**: 베트남어+영어, 한국어+영어 | 번역 + 응답에 이중 시간 | 🟡 중간 |
| **야간/비활성 시간대**: 라이브 후 댓글 방치 | 골든타임 응답 실패 | 🟡 중간 |

### 셀러 시간 분석 (추정)
- 소규모 셀러 (50~200 댓글/일): **1~2시간/일** 댓글 관리
- 중규모 셀러 (200~1,000 댓글/일): **3~5시간/일** 또는 전담 인력 필요
- 대규모 셀러 (1,000+ 댓글/일): CS 팀 운영 필요 (**월 $500~2,000 인건비**)

## 4. TAM / SAM / SOM 추정

### TAM (Total Addressable Market) — 전체 시장
> TikTok Shop 활성 셀러 × 연간 댓글 관리 도구 지출 의향

- 글로벌 TikTok Shop 활성 셀러: ~200만 명 (추정)
- 평균 연간 도구 지출: $240/년 ($20/월)
- **TAM = ~$480M/년**

### SAM (Serviceable Addressable Market) — 접근 가능 시장
> 베트남 + 한국(진입 시) + 동남아 뷰티/패션 셀러

- 베트남 TikTok Shop 활성 셀러: ~30만 명
- 한국 TikTok 크리에이터 셀러 (Shop 출시 시): ~5만 명
- 뷰티/패션 비중: ~40%
- 대상 셀러: ~14만 명
- **SAM = ~$33.6M/년**

### SOM (Serviceable Obtainable Market) — 획득 가능 시장
> 첫 2년간 현실적 목표

- 목표 가입자: 1,000명 (Year 1) → 5,000명 (Year 2)
- 유료 전환율: 15%
- ARPU: $20/월
- **SOM Year 1 = ~$36K/년 (MRR $3K)**
- **SOM Year 2 = ~$180K/년 (MRR $15K)**

## 5. TikTok API 현황

### Content Posting API
| 항목 | 현황 |
|------|------|
| 영상 게시 | ✅ Direct Post (v2/post/publish/video/init/) |
| 크리에이터 정보 조회 | ✅ creator_info/query |
| 댓글 읽기 | ⚠️ 공식 API 제한적 — Research API로 일부 가능 |
| 댓글 쓰기 (대댓글) | ❌ 공식 API 미지원 |
| OAuth | ✅ 사용자 인증 + access_token |
| Rate Limit | 6 req/min/user (영상 게시 기준) |
| 감사 요구 | ✅ 미감사 클라이언트는 private 모드 제한 |

### 댓글 관련 API 상세

#### 읽기 (가능)
- **TikTok Research API**: 학술/리서치 목적으로 댓글 데이터 접근 가능
- **Display API**: 임베드용 영상 정보 (댓글 포함 제한적)
- **비공식 방법**: 웹 스크래핑 (ToS 위반 리스크)

#### 쓰기 (제약)
- **공식 댓글 작성 API 없음**: Content Posting API는 영상 게시만 지원
- **대안 1**: 셀러가 Tikly에서 생성된 답변을 복사 → TikTok 앱에 붙여넣기
- **대안 2**: 브라우저 확장 프로그램으로 TikTok 웹에서 자동 입력
- **대안 3**: TikTok API 업데이트 대기 (댓글 API 추가 가능성)

### API 활용 전략 (MVP)
```
Phase 1 (MVP): 
  - 댓글 읽기: Research API 신청 or Display API 활용
  - 댓글 쓰기: 수동 복사-붙여넣기 (AI가 초안만 생성)
  - 가치 제안: "분석 + 초안 생성"에 집중

Phase 2:
  - 댓글 읽기: Webhook 기반 실시간 수집 (API 업데이트 시)
  - 댓글 쓰기: 브라우저 확장 프로그램 (Chrome Extension)
  
Phase 3:
  - 공식 댓글 API 출시 시 네이티브 연동
```

### 주요 제약사항 정리
1. **댓글 쓰기 API 부재**: MVP 핵심 리스크 → "분석+초안" 가치로 우회
2. **Research API 접근 제한**: 승인 프로세스 필요 (학술/비즈니스 목적 구분)
3. **Rate Limit**: 대량 처리 시 큐잉 필수
4. **Business 계정 전용**: 개인 계정 셀러 온보딩 마찰
5. **지역 제한**: 일부 API 기능이 특정 국가에서만 가능

## 6. 시장 진입 전략

### Phase 1: 베트남 뷰티 셀러 (0~6개월)
- 베트남 TikTok Shop $4.5B 시장, 뷰티 카테고리 Top 3
- 초기 50명 베타 셀러 모집 (라이브커머스 셀러 우선)
- 베트남어 감성분석 정확도 집중 투자

### Phase 2: 한국 크리에이터 셀러 (6~12개월)
- TikTok Shop 한국 출시 시점에 맞춰 진입
- 한국 뷰티/패션 인플루언서 타겟
- K-뷰티 해외 판매 셀러 (글로벌 TikTok Shop) 선공략

### Phase 3: 동남아 확장 (12~18개월)
- 태국, 인도네시아, 필리핀 순차 진출
- 현지 언어 지원 추가
