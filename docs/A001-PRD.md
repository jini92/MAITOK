# A001 — Tikly PRD (Product Requirements Document)

## 1. 제품 비전 & 미션

### 비전
> TikTok 셀러가 댓글 하나도 놓치지 않고, AI로 매출 기회를 극대화하는 세상

### 미션
TikTok Shop 셀러에게 AI 기반 댓글 감성분석과 자동 대댓글 서비스를 제공하여, 댓글 관리 시간을 90% 줄이고 구매 전환율을 높인다.

### 핵심 가치 제안
- **시간 절약**: 수백 개 댓글을 수동 확인하는 대신, AI가 우선순위를 매기고 답변 초안을 생성
- **매출 증대**: 구매의도 댓글("얼마에요?", "배송 며칠?")을 실시간 감지하여 즉시 응대
- **리스크 관리**: 부정 댓글/클레임을 조기 포착하여 브랜드 위기 방지

## 3. 핵심 기능 상세

### 3.1 댓글 감성분석 (Sentiment Analysis)
- **분류 카테고리**: 긍정 / 부정 / 질문 / 구매의도 / 스팸
- **구매의도 감지**: "가격", "얼마", "배송", "사이즈", "재입고" 등 키워드 + 문맥 분석
- **다국어 지원**: 한국어, 베트남어, 영어
- **실시간 처리**: 새 댓글 5분 이내 분류 완료
- **대시보드**: 감성 분포 차트, 시계열 추이, 핫 키워드 워드클라우드

### 3.2 자동 대댓글 생성 (Auto-Reply)

#### 반자동 모드 (Suggested Reply)
1. AI가 댓글 분석 → 대댓글 초안 생성
2. 셀러에게 Discord/Telegram 알림 (댓글 원문 + AI 초안)
3. 셀러가 승인/수정 → 게시

#### 완전자동 모드 (Auto-Reply)
1. FAQ 패턴 매칭 시 자동 게시 (셀러가 사전 설정한 규칙 기반)
2. 구매의도 댓글 → 상품 링크 포함 자동 응답
3. 스팸 댓글 → 자동 숨김/신고

#### 응답 톤 설정
- 브랜드별 톤 커스터마이징 (친근함 / 전문적 / 캐주얼)
- 이모지 사용 빈도 설정
- 금지 표현 필터

### 3.3 셀러 리포트 (Analytics Report)

#### 일일 리포트 (Daily Digest)
- 총 댓글 수 / 감성 분포 (긍정:부정:중립 비율)
- 구매의도 댓글 수 + 응답률
- 핫 키워드 Top 10
- 주의 필요 댓글 하이라이트

#### 주간 리포트 (Weekly Summary)
- 주간 트렌드 변화
- 경쟁사 대비 응답 속도 벤치마크
- 구매 전환 기회 분석

## 4. 사용자 플로우

### 4.1 온보딩 (Onboarding)
```
1. Tikly 웹/앱 가입
2. TikTok Business 계정 OAuth 연동
3. 모니터링할 영상/계정 선택
4. 응답 톤 & 자동화 규칙 설정
5. 알림 채널 연결 (Discord/Telegram)
6. 첫 번째 분석 리포트 수신 (10분 내)
```

### 4.2 일상 사용 (Daily Flow)
```
아침: 일일 리포트 수신 → 주의 댓글 확인
     ↓
일중: 구매의도 댓글 알림 → 반자동 모드로 빠른 응답
     ↓
자동: FAQ 댓글 자동 응답 (설정된 규칙)
     ↓
저녁: 라이브 후 폭주 댓글 → AI 일괄 분류 + 초안 생성
     ↓
주간: 주간 리포트로 전략 조정
```

## 5. MVP vs v2 기능 구분

### MVP (v1) — 출시 목표: 3개월
| 기능 | 상세 |
|------|------|
| TikTok 댓글 수집 | 특정 영상 댓글 polling (Content Posting API) |
| AI 감성분석 | 긍정/부정/질문/구매의도/스팸 5단 분류 |
| 반자동 대댓글 | AI 초안 → 셀러 승인 → 게시 |
| 일일 리포트 | Discord DM으로 일일 요약 발송 |
| 단일 언어 | 한국어 우선 |
| 알림 | Discord 봇 알림 |

### v2 — MVP 출시 후 3개월
| 기능 | 상세 |
|------|------|
| 완전자동 모드 | FAQ 패턴 매칭 자동 응답 |
| 다국어 지원 | 한국어 + 베트남어 + 영어 |
| 웹 대시보드 | 감성 트렌드, 키워드 분석, 응답 관리 |
| Telegram 알림 | Discord 외 Telegram 봇 지원 |
| 주간 리포트 | 트렌드 분석 + 벤치마크 |
| 다계정 관리 | 복수 TikTok 계정 통합 관리 |

### v3 — 장기 로드맵
- TikTok Shop 주문 연동 (댓글 → 주문 전환 추적)
- Instagram Reels / YouTube Shorts 확장
- SaaS 대시보드 (멀티테넌트)
- 팀 협업 기능 (역할 기반 접근 제어)

## 7. 경쟁 분석

### 경쟁 서비스 비교

| 기능 | **Tikly** | ManyChat | Chatfuel | 수동 관리 |
|------|-----------|----------|----------|-----------|
| TikTok 댓글 감성분석 | ✅ AI 5단 분류 | ❌ | ❌ | ❌ |
| 구매의도 자동 감지 | ✅ | ❌ | ❌ | 수동 |
| 댓글 자동 대댓글 | ✅ 반자동+완전자동 | ❌ (DM만) | ❌ (DM만) | 수동 |
| TikTok DM 자동화 | 🔜 v2 | ✅ | ✅ | 수동 |
| 일일 리포트 | ✅ Discord/Telegram | ❌ | ❌ | ❌ |
| 다국어 (한/베/영) | ✅ v2 | 부분적 | 부분적 | N/A |
| 가격 | $10~30/월 | $15~65/월 | $15~45/월 | 인건비 |
| TikTok Shop 특화 | ✅ 핵심 | ❌ 범용 | ❌ 범용 | N/A |

### Tikly 핵심 차별점
1. **댓글 중심**: ManyChat/Chatfuel은 DM 자동화에 집중, Tikly는 **공개 댓글** 분석+응답에 특화
2. **감성분석 내장**: 경쟁사는 키워드 매칭만, Tikly는 LLM 기반 문맥 이해
3. **구매의도 감지**: 댓글에서 매출 기회를 자동 포착하는 유일한 도구
4. **셀러 리포트**: 경쟁사에 없는 일일/주간 인사이트 리포트
5. **아시아 시장 특화**: 한국어/베트남어 네이티브 지원

## 8. 기술 아키텍처 (개요)

```
TikTok API (Content Posting API)
    ↕ polling / webhook
Tikly Backend (Node.js)
    ├── Comment Collector (댓글 수집)
    ├── AI Engine (OpenAI/Claude → 감성분석 + 답변생성)
    ├── Reply Manager (대댓글 큐 + 게시)
    ├── Report Generator (일일/주간 리포트)
    └── Notification Service (Discord/Telegram 봇)
         ↕
    Seller Interface (Discord 봇 / 웹 대시보드 v2)
```

### 기술 스택
- **Backend**: Node.js + TypeScript
- **AI**: OpenAI GPT-4o / Claude (감성분석, 답변생성)
- **DB**: PostgreSQL (댓글 데이터) + Redis (큐/캐시)
- **API**: TikTok Content Posting API
- **알림**: Discord.js / Telegram Bot API
- **인프라**: Cloudflare Workers / Railway

## 9. 리스크 & 완화 전략

| 리스크 | 영향 | 완화 전략 |
|--------|------|-----------|
| TikTok API 댓글 쓰기 미지원 | 핵심 기능 불가 | 우회: 셀러가 앱에서 복사-붙여넣기 / 브라우저 확장 |
| API Rate Limit (6 req/min/user) | 대량 처리 지연 | 큐 기반 배치 처리 + 우선순위 큐 |
| AI 오답 (감성 오분류) | 부적절 자동 응답 | MVP는 반자동 모드 기본, 완전자동은 신뢰도 threshold |
| TikTok 정책 변경 | 서비스 중단 위험 | 멀티 플랫폼 확장으로 의존도 분산 |
| 베트남어 AI 정확도 | 현지 셀러 만족도 저하 | 베트남어 fine-tuning 데이터 수집 + 피드백 루프 |
